Scenario Management

Question: How to create a new Asset?

To create a new Asset and a Project, Click "Scenario Manager" from Precept main User Interface.

By default, you will be the only person who can work on this new Asset. To allow team members, to work on that Asset, you need to click 'Security" from Precept Main User Interface and add them.

 

 

Question: I cannot see the Asset my team members are working on?

You need to be added to the list of approved Users for that Asset. You can ask the member who created that Asset or a "Super-User" to add you, from the "Security" interface.

 

 

Question: I cannot see the Scenario my team members is working on?

It could be that the scenario has not been shared by the use who created it. Scenario status can be set to 'shared" or "Private" in the "Scenario Manager" interface.

 

 

Question: How to create a new Scenario?

If you have already created an Asset and a Project, you can create  new scenario by clicking "Scenario Manager" from Precept main User Interface. Then you ahve two options.

 Option 1: Create a scenario from scratch by clicking "Create new scenario from scratch"

 Option 2: Duplicate a scenario from an existing scenario by clicking "Import / Duplicate scenario"

 

 

 

 

Modeling Oil and Gas Assets

 

Question: Why do I need to give a "Project" name when I introduce a Shutdown?

You are asked to provide a "Project" name when introducing a Shutdown for the following reasons:

1.Tracking and organization: Each Shutdown can be linked to a specific Project (for example: maintenance, Reservoir Management activities). The Project can be included or excluded of a scenario from the "Settings" interface.

2.When running forecast, shortfalls are calculated by Precept and grouped by "Project". Then, in "Results" section, along with actual production, you can select the "Project" and get the associated shortfalls.

 

Question: Why do I need to give a "Project" name when I introduce a "new Profile" ?

In Precept, you are required to provide a "Project" name when introducing a new Profile for these main reasons:

          1. Organization and traceability: Linking a new Profile to a specific project helps you keep track of why and for which context the Profile was created (for example: a drilling campaign (phase 1, phase2), or a stimulation campaign, etc...)

          2. In "Results" section, Forecast can be seen and grouped by "Project". So you can. for example,  visualize the producrion coming from the existing well stocks without further action, and get the incremental coming from Phase 1 drilling, Phase 2 drilling,                     stimulation campaign, etc...).

 

 

 

 

Decline Curves Module:

 

Question: In the DCA module, how can I compare the potential of a well with its nearby wells?

You can add in the background of the DCA charts the potentials of one or several nearby wells by selecting the Scenario and Wells of interests, using the "Plot in background" menu on the left side of the interface.

 

 

Question: I have introduced a "Nominal Decline" and "b factor" for a well. How can I know the corresponding effective decline?

In the DCA module, click "Settings" on the main chart and toggle-on the option "Decline rate (%). You can then visualize on the chart the actual yearly decline.

 

 

Question: in the DCA module, for an individual well, can Precept generate for me a Low case and High case from a Mid case?

In the DCA module, click "Settings" on the main chart. You can then define a ratio for the cumulative production 3P/2P and 1P/2P.

Then Click on "Auto-forecast Low / High from Mid". Precept will then play with "Nominal decline" and "b factor" to generate low and high cases potentials that honors the ratio 3P/2P and 1P/2P.

 

 

Question: In the DCA module, can I remove outliers historical production data before asking Precept to generate well potentials using auto-forecast?

Yes, you can remove outliers historical production data by clicking "Unselect" and "create a box" around the values to be filtered-out.

 

 

Question: In the DCA module, can I select a certain period of time to be used to generate well potentials using auto-forecast?

Yes, you can select a certain period of time by clicking "Select" and "create a box" around the values to be used for Auto-forecast.

 

 

Question: In the DCA module, can I change the default charts provided by Precept ?

yes. The two charts on the right of the interface can be modified. make sure that the drop-down menu on the top right of the chart is set to "Plot DCA". You can then change the parameters you want to plot (Oil, water, Gas, Water Injection, etc..) but also the X-Axis.

X-Axis can be set to Date, Cumulative Oil, Production time (years) or even Cum Oil N/Np.

 

 

Question: In the DCA module, the charts are too busy with many curves, how to make them easier to read?

You can click on "Settings" and toggle on/off some parameters. For example, if you want to focus on oil potential, you can toggle-off the liquid and gas parameters.

 

 

Question: Can I compare the DCA potentials of an individual well with the potentials coming from Reservoir Simulator like Eclipse or Intersect?

Yes. In order to do that, please follow the steps below:

1. Duplicate the current scenario and give a name to the new scenario like "Simulation results".

2. Go to "Decline curves', then click on "Import / Duplicate profile"

3. Load the simulation results, either by direct import from Eclipse or Intersect RSM file, or import from Excel (assuming you already have all the simulations profiles in Excel format)

4. in the DCA module, using the "Plot in background" menu, select the "Simulation results" scenario and the well to be plotted in the background.
