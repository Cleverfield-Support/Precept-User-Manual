FAQ

 

 

 

 

Question: How to create a new Asset?

To create a new Asset and a Project, Click "Scenario Manager" from Precept main User Interface.

By default, you will be the only person who can work on this new Asset. To allow team members, to work on that Asset, you need to click 'Security" from Precept Main User Interface and add them.

 

 

Question: I cannot see the Asset my team members are working on?

You need to be added to the list of approved Users for that Asset. You can ask the member who created that Asset or a "Super-User" to add you, from the "Security" interface.

 

 

Question: I cannot see the Scenario my team members is working on?

It could be that the scenario has not been shared by the use who created it. Scenario status can be set to 'shared" or "Private" in the "Scenario Manager" interface.

 

 

Question: How to create a new Scenario?

If you have already created an Asset and a Project, you can create  new scenario by clicking "Scenario Manager" from Precept main User Interface. Then you ahve two options.

 Option 1: Create a scenario from scratch by clicking "Create new scenario from scratch"

 Option 2: Duplicate a scenario from an existing scenario by clicking "Import / Duplicate scenario"
