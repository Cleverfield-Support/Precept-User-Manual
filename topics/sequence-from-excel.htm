Build sequence from Excel

 

 

 

 

Precept allows user to build a sequence like a drilling schedule for a Rig by importing that sequence from Excel.

 

1. From Precept Main User Interface, click Sequence then "Import schedule from Excel"

2. Select the string you want to import the Schedule (ex: Rig 1, Rig 2)

3. Copy-paste the Schedule from Excel using the format below (Column order needs to start from 1)

 

Order

Opportunity

1

OP01

2

OP02

3

OP03

4

OP04

 

 

 

 

 

4. Click "Save sequence order"
